### 技术细节

#### FFmpeg 滤镜说明

文档详细解释了双层进度条的实现原理：

- **底层（背景条）**：静态灰色条，使用 `drawbox` 滤镜绘制全宽矩形
- **上层（前景条）**：动态进度条，使用表达式 `w='iw*t/<duration>'` 实现宽度随时间增长

#### 参数表格

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| input | - | 输入视频文件路径（必需） | - |
| --output | -o | 输出视频文件路径 | <input>_with_progress.<ext> |
| --position | -p | 进度条位置（top 或 bottom） | bottom |
| --height | - | 进度条高度（像素） | 5 |
| --bg-color | - | 背景颜色（十六进制） | 808080 |
| --fg-color | - | 前景颜色（十六进制） | FF0000 |
